# การนำไปใช้งานบน Synology NAS

คู่มือนี้จะอธิบายวิธีการรัน `free-qrcode` Docker image บน Synology NAS โดยใช้ไฟล์ `.tar` ที่ export มา

## สิ่งที่ต้องมี

*   **Synology NAS** ที่ใช้ DSM 7.0 ขึ้นไป (แนะนำ)
*   แพ็กเกจ **Container Manager** (ชื่อเดิมคือ "Docker") ติดตั้งจาก Package Center
*   ไฟล์ `free-qrcode-amd64.tar` ที่คุณ export ออกมา (รองรับ AMD64 สำหรับ Synology)

## ขั้นตอนที่ 1: อัปโหลด Image

1.  เปิด **File Station** บน Synology NAS ของคุณ
2.  สร้างโฟลเดอร์ (เช่น `docker/free-qrcode`) หรือไปยังตำแหน่งที่คุณต้องการเก็บไฟล์
3.  อัปโหลดไฟล์ `free-qrcode-amd64.tar` จากคอมพิวเตอร์ของคุณไปยังโฟลเดอร์ที่เตรียมไว้

## ขั้นตอนที่ 2: Import Image

1.  เปิด **Container Manager** (หรือ "Docker")
2.  ไปที่แท็บ **Image**
3.  คลิก **Action** > **Import** > **Add from File**
4.  เลือกไฟล์ `free-qrcode.tar` ที่คุณเพิ่งอัปโหลด
5.  คลิก **OK** ระบบจะทำการโหลด image เมื่อเสร็จสิ้น คุณควรจะเห็น `meen2nont/free-qrcode:latest` ปรากฏขึ้นในรายการ

## ขั้นตอนที่ 3: รัน Container

1.  เลือก image `meen2nont/free-qrcode:latest` ในรายการ
2.  คลิก **Run**
3.  **General Settings (การตั้งค่าทั่วไป)**:
    *   **Container Name**: ตั้งชื่อคอนเทนเนอร์ เช่น `free-qrcode`
    *   **Enable auto-restart**: ติ๊กถูกถ้าต้องการให้แอปเริ่มทำงานอัตโนมัติเมื่อ NAS รีบูต
    *   คลิก **Next**
4.  **Port Settings (การตั้งค่าพอร์ต)**:
    *   คุณต้องจับคู่พอร์ตบน NAS กับพอร์ตเว็บเซิร์ฟเวอร์ของคอนเทนเนอร์ (80)
    *   **Local Port**: เลือกพอร์ตที่ว่างอยู่ เช่น `8080`
    *   **Container Port**: `80` (ห้ามเปลี่ยนค่านี้)
    *   ตัวอย่าง: `8080` -> `80` หมายความว่าคุณจะเข้าใช้งานผ่าน `http://IP_ของ_NAS:8080`
5.  **Volume Settings** (ไม่บังคับ):
    *   แอปนี้เป็นแบบ static ไม่จำเป็นต้องตั้งค่า volume เว้นแต่คุณจะมีการแก้ไข config ขั้นสูง สามารถข้ามขั้นตอนนี้ได้
6.  คลิก **Next**, ตรวจสอบข้อมูลสรุป, แล้วคลิก **Done**

## ขั้นตอนที่ 4: เข้าใช้งานแอป

1.  เปิดเว็บเบราว์เซอร์บนคอมพิวเตอร์ของคุณ
2.  ไปที่ `http://IP_ของ_NAS:PORT_ที่คุณตั้ง` (ตัวอย่างเช่น `http://192.168.1.100:8080`)
3.  คุณควรจะเห็นหน้าเว็บ QR Code Generator พร้อมใช้งาน!
